twitterspy by dustin

TwitterSpy is a supplemental XMPP bot for twitter that does the stuff the twitter one used to do, and a few things it didn't.

Dependencies

  • Twisted
    • names
    • web
    • words
  • SQLAlchemy

Install

  1. Install dependencies
  2. git submodule init && git submodule update
  3. copy twitterspy.conf.sample to twitterspy.conf
  4. edit twitterspy.conf
  5. ./etc/create_tables.py
  6. twisted -ny twitterspy.tac
